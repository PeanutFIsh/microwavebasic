2. Command reference

To correctly understand some of the commands, I recommend you to take a look at the Chapter 3 !
The Chapter 2 is about all the built-in commands with examples of each of these. (I love examples when I have a reference so I am gonna write my reference out of examples)
P (string) || p (string)

Those two commands will allows you to display a string expression

The difference between p and P is that P prints the string with a linebreak unlike p

Example :
10 p "Hello, "
20 P " World!"
The output is gonna be Hello, World! with a line break at the end.
?(*/$) (var)

Inputs what you type on the keyboard.

Example :
// Inputs a string then print it
10 ?$ A
20 P A

// Inputs an integer then print it
10 ?* A
20 * A
30 P ""
// (COMMENT)

Example :

Prints a math expression

Example :
10 p "2 plus 2 is "
20 * 2 + 2
30 p " minus one is "
40 * 4 - 1
// Quick maths!
50 P ""
=(*/$) (variable) (expression)

Sets a value to a specific variable.

Example :
10 =* A 2+2-1
20 =$ A "2 plus 2 is 4 minus 1 is"
30 p A
40 * A
50 " Quick maths!"
// String and Int variables are separated.
G (line)

Go to a specific line of the program

Example :

I($/*) (condition):(line of code)

If (condition) then execute (line of code)

Example :

40 p "Type 69: "

3. Expression system

The expression system is just a system to do... uhm... simple math, string and condition expressions.
Math expressions

The Math expression system is extremely simple and limited. You don't have parentheses, can only operator with some basic operator (+ - * /) and works with variables
For example, 2 + 2 - A is a valid expression. However, 2 + (2 - A) is unfortunealy an invalid expression. This is planned to be added later.

Math expressions can be used anywhere it asks an integer expression, including conditions.
String expressions

Like the math one, the String expression system is also simple and limited. You can't do anything more than string addition like this : "Hello, " + "World!", same as the math expressions, it supports variables.

Like math expressions, it can be used anywhere it asks a string expression, also including conditions.
Condition expressions

The condition expression system is nothing more than uhm.. conditions. And, again, very simple and limited.
Boolean operators for strings are = and !
Boolean operators for integers are = > < !

= is Equal, will return true if the value A and the value B are equal.
Example : I$ A="henlo":P "henlo frend" will print "henlo frend" if the variable A is equal to "henlo"

Works for strings and integers.
! is the opposite of Equal, inequal. Returns true if Value A and Value B are inequal

Works for strings and integers.
> and < are less than and more than operators

Example :
5 p "Please tell me the hour :"
10 ?* A
20 I* A < 12:P "Good morning!"
30 I* A > 12:I* A < 18:P "Good afternoon!"
40 I* A > 18:P "Good evening!"

Also there isn't any logical operators, sorry
