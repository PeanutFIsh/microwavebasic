2. Command reference

To correctly understand some of the commands, I recommend you to take a look at the Chapter 3 !
The Chapter 2 is about all the built-in commands with examples of each of these. (I love examples when I have a reference so I am gonna write my reference out of examples)
P (string) || p (string)

Those two commands will allows you to display a string expression

The difference between p and P is that P prints the string with a linebreak unlike p

Example :
10 p "Hello, "
20 P " World!"
The output is gonna be Hello, World! with a line break at the end.
?(*/$) (var)

Inputs what you type on the keyboard.

Example :
// Inputs a string then print it
10 ?$ A
20 P A

// Inputs an integer then print it
10 ?* A
20 * A
30 P ""
// (COMMENT)
